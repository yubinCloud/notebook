## 1. MySQL 是怎么保证主备一致的？

MySQL 能够成为现下最流行的开源数据库，binlog 功不可没，binlog 可以用于归档，也可以用来做主备同步。MySQL 在最开始以方便的高可用架构而被青睐，而它几乎所有的高可用架构都直接依赖于 binlog。虽然这些高可用架构已经呈现出越来越复杂的趋势，但都是从最基本的一主一备演化过来的。

这一章将介绍一下主备的基本原理以及设计思想。
